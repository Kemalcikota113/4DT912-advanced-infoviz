## How to run (linux)

1. cd in to directory
```
cd (path)/A1/
```

2. create venv and activate
```
python3 -m venv venv
source venv/bin/activate
```

3. pull dependencies
```
pip install -r requirements.txt
```

4. start script
```
python3 visualize_publications.py
```
